Real-Time Weather & Air Quality Monitor

It is a comprehensive weather dashboard designed to provide users with hyper-local environmental data. 
Beyond standard temperature tracking, the application integrates Air Quality Index (AQI) and detailed atmospheric metrics to give a full picture of the user's surroundings.

🚀 Features

Live Weather Updates: Real-time temperature, "Feels Like" conditions, and weather descriptions.

Atmospheric Data: Detailed breakdown of humidity, pressure, and wind speed.

Geospatial Tracking: Precise longitude and latitude coordinates for the searched location.

Air Quality Index (AQI): Monitoring of pollutants (PM2.5, PM10, $NO_2$, etc.) to ensure health safety.

Dynamic Backgrounds: UI changes based on the current weather condition (Clear, Rainy, Snowy).# Weather-application-for-live-forecasting.
